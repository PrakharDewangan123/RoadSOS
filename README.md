# 🚨 RoadSoS – Road Accident Emergency Web App

RoadSoS is a modern road accident emergency response web application built with React + Vite.  
It provides instant SOS actions, real-time location sharing, emergency service mapping, and quick access to hospitals, police stations, and vehicle repair services.

This project is fully client-side and ready to run, share, or submit for hackathons.

---

## 🚀 Features

### 🚨 Emergency System
- **SOS Button**
  - One-tap emergency alert
  - Logs activity and triggers emergency workflow
  - Opens Google Maps with user location

- **Accident Mode (🔥 Highlight Feature)**
  - Activates:
    - SOS alert  
    - Location detection  
    - Nearby hospital search  
  - All in one click  

---

### ⚡ Quick Actions
- **Hospital Finder** – Opens nearest hospitals  
- **Police Access** – Finds nearby police stations  
- **Vehicle Repair** – Finds nearby mechanics  
- **Share Location** – Copies Google Maps link  

---

### 🗺️ Emergency Map System
- Built using Leaflet + OpenStreetMap  
- Displays:
  - 🏥 Hospitals  
  - 🚔 Police Stations  
  - 🔧 Repair Services  
- Shows user’s real-time location  
- Provides navigation via Google Maps  

---

### 📍 Location System
- Uses Geolocation API  
- Gets real-time coordinates  
- Enables accurate emergency response  

---

### 📞 Emergency Contacts
- Store important contacts  
- Quick access during emergencies  

---

### 📊 Activity Tracking
- Logs SOS alerts  
- Tracks emergency actions  

---
